# Projeto QR code

Página QR code responsiva, para DESKTOP e MOBILE, apenas com html e css.

## Tecnologias

- HTML
- CSS

## Instalação

1. Clone o repositório: `git clone https://github.com/SouDann/QR-code.git`
2. Abra o arquivo `index.html` no seu navegador.
